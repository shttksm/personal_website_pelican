Pelicanでgit.ioにアップロードするやり方をメモ

  • とりあえずGitのページからioが使える設定に変更

ページの雛形作成

  • pelican-quickstart

記事を書く

  • content/ の配下に.mdファイルで記事ページを書く
  • make html でoutput/配下にmdから変換されたhtmlファイルが保存
  • ローカルで表示確認 make serve, http://localhost:8000/ で確認

テーマ変更

  • 対象のgit ディレクトリにテーマのgitリポからgit clone してきてコピー

    • http://pelicanthemes.com/

  • THEME = './nest' pelicanconf.py のテーマを取ってきたテーマ名とする．

    • ブログ名など種々の設定もこのpyファイルをいじる．

公開

  • pelican content -o output -s pelicanconf.py
  • ghp-import output -b gh-pages
  • git push origin gh-pages

ghp-import はlocal master で編集していたとして，公開する際に別ブランチのgh-pagesのoutputディレクトリにhtmlファイルを作ってステージングとコミットする．そんで，pushするときにremoteのorigin にローカルのgh-pagesを上書きする．
